Meu Primeiro Projeto em JavaScript

Publicado em 15 de Junho de 2025 | Categoria: Programação | Autor: Igor Yoshimitsu Ide

Recentemente comecei a aprender JavaScript e decidi criar meu primeiro projeto: um simples jogo da velha. Foi desafiador, mas extremamente gratificante ver o código funcionando.

Por que um Jogo da Velha?

Escolhi começar com um jogo da velha porque:

  • É simples o suficiente para um primeiro projeto
  • Envolve lógica de programação básica
  • Permite praticar manipulação do DOM
  • Tem um escopo bem definido

Desafios Encontrados

Alguns dos desafios que enfrentei:

// Exemplo de código do projeto
function checkWinner() {
    const winningCombinations = [
        [0, 1, 2], [3, 4, 5], [6, 7, 8], // linhas
        [0, 3, 6], [1, 4, 7], [2, 5, 8], // colunas
        [0, 4, 8], [2, 4, 6]             // diagonais
    ];
    
    for (let combo of winningCombinations) {

        if (board[a] && board[a] === board[b] && board[a] === board[c]) {
            return board[a];
        }
    }
    return null;
}

Lições Aprendidas

Este projeto me ensinou várias coisas valiosas:

  1. Como estruturar um projeto simples
  2. A importância do planejamento antes de codar
  3. Como depurar código JavaScript
  4. Manipulação básica do DOM
